# Features

- Hacker News Screen

  - We have used [Hacker News API](https://github.com/HackerNews/API) for display list of hacker news.

- Comment News Screen

  - We have displayed news commnet with SentimentAnalysis Count using NaturalLanguage's NLTagger FrameWork.
